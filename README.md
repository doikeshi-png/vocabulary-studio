# Vocabulary Studio — fixed modes

Главный файл для GitHub Pages: `index.html`.

Что исправлено:
- German-only режимы: Article и 3 verb forms.
- Synonyms, Collocations, Cloze и Writing работают от отдельных полей карточки.
- Письменный режим показывает, что именно нужно написать.
- Интерфейс переключается: RU / UKR / DE / EN.
- Колоды сохраняются в браузере через localStorage.

Форматы:

English:
```txt
word | synonyms | collocations | note | example
```

Deutsch noun:
```txt
article | word | English meaning | collocations / grammar pattern | note | example
```

Deutsch verb:
```txt
 | nehmen / nahm / genommen | to take | den Zug nehmen; etwas mitnehmen | strong verb | Ich nehme heute den Zug.
```
